GRDB Version 2.7.1

A Useless Safari extension.

List messages, contacts, and visitors in a more concise and clear way.

New in this Version

The user interface has been redesigned to be more clear.

Included in Version 2.6

You can now drag images from the site or use the context menu to save them.

User icons will be shown in the visitor list and now also for favourite contacts.

Include distance in visitor and search lists.

Footprints for special events are now visible in the visitor list.

The new and improved search option makes it even easier to find exactly what you are looking for.

Display counts of new items in header and popover.

Expand actions for list items.

Toolbar Button

Clicking the toolbar button opens the homepage where you can sign in.

When signed in, the number of new messages is badged on the toolbar icon. Click the toolbar button to show the message list.

Click an hold the toolbar button to see the popover with your sign-in state and item counts. New items are indicated to the left of the overall count.

    Pinned user

You can pin visitors to your list. So they keep being displayed in this list even after they disappear from the website visitor list.

Hover over some elements to display further information. Such as profile picture, user stats, footprint title, message age, or contact note.

Icons

Several icons are used throughout the extension.

Actions

These icons are placed to the left of list items. Click them to perform the described action.

  • Show history
  • Show history (with messages)
  • Forward history via e-mail
  • Compose follow-up on sent message
  • Compose new message
  • Read latest message from user
  • Show more actions
  • Save or edit contact
  • Show picturea album
  • Leave a footprint
  • Keep visitor in list
  • No longer keep visitor in list

Status Icons

These icons appear after profile names when there online status is known.

  • Online
  • Away
  • Offline
  • Deleted

Footprints

In the visitor list, footprints are displayed with different shapes and icons.

  • You left a footprint on another profile
  • You received a footprint from a visitor
  • You and a visitor left each other footprints
  • You left each other matching footprints
  • You left “thank you” on a received footprint

The icon within the footprint shape hints at the kind of footprint you left or received.

  • General
  • Date
  • Explicit
  • Greetings
  • Sexy
  • Compliments
  • Love
  • NO
  • Thank you

Report a Problem

We depend on a certain structure of the GR website. When there are some changes on their side, some things may break.

There may be other things that went unnoticed. If you find something that isn’t working as expected, don’t hesitate to report on our issue page.
